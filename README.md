# odin-recipes
## Recipes project from the Odin Lessons

- Version en Español
Este es el primer proyecto del curso de Odin Project, para esto, en vez de realizar un pagina con recetas de cocina, voy a realizar un pequeño recetario de recetas de pintura de miniaturas.

Si bien no sera un blog, sera la primera version de un recetario para los que estan adentrandose en el mundo de la pintura de miniaturas ya sea para wargaming, DND, o gaming.

- English Version
This is the first project for the Odin Project course. Instead of creating a website with cooking recipes, I'm going to create a small recipe book for miniature painting.

While it won't be a blog, it will be the first version of a recipe book for those who are getting into the world of miniature painting, whether for wargaming, D&D, or gaming.
